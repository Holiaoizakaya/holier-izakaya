# LINE 官方帳號優化設定

這是一個不需安裝套件的靜態網頁工具，用來檢查 LINE 官方帳號上線前與營運中的基礎設定。

## 啟動

在此資料夾執行：

```bash
python3 -m http.server 8000
```

開啟 `http://localhost:8000`。

## 功能

- 依「基本設定、訊息體驗、好友導流、資料追蹤」分類檢核
- 勾選進度會保存在瀏覽器 `localStorage`
- 一鍵標記必要項目完成
- 匯出 CSV 清單，方便交接或留存

## 使用建議

先完成標示為「必要」的項目，再依實際營運情境補齊「建議」項目。每週回頭檢查好友數、封鎖數、訊息點擊與導流成果，依數據調整歡迎訊息、圖文選單與推播內容。
